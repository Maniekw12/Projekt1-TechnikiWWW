T9 Simulator

Author: Marian Wachała

About the T9 System

T9 is a predictive text input technology designed for mobile phones with a 3×4 numeric keypad. It was originally developed by Tegic Communications, now part of Nuance Communications. The name "T9" stands for Text on 9 keys.

History and Usage

In the late 1990s, T9 was used in phones by manufacturers such as Verizon, NEC, Nokia, Samsung, Siemens, Sony Mobile, Sanyo, and SAGEM, as well as PDAs like Avigo. Key competitors included Motorola's iTap, RIM's SureType, Eatoni's LetterWise and WordWise, and Intelab's Tauto.

Availability Today

Today, T9 is still available on niche devices like the Punkt mp-02, on some non-touchscreen phones, and as a custom keyboard for Android and iPhone devices (from iOS 18).

How T9 Works

  • Each numeric key (2-9) represents 3-4 letters
  • The system predicts words based on the sequence of key presses
  • It uses a dictionary to suggest the most likely words
  • Users can cycle through suggestions using the * or # keys

Technologies

This project is written in Python using Trie data structure.

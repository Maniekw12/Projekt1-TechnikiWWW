T9 Simulator

Author: Marian Wachała

About T9 Simulator

T9 is a predictive text input technology designed for mobile phones with a 3×4 numeric keypad. It was originally developed by Tegic Communications, now part of Nuance Communications. The name "T9" stands for Text on 9 keys. This simulator recreates the classic T9 experience and demonstrates how predictive text technology revolutionized mobile communication in the pre-smartphone era.

Features of T9 System

  • Predictive text input using numeric keypad (2-9 keys)
  • Dictionary-based word suggestions for faster typing
  • Cycle through word suggestions using special keys
  • Efficient text entry with minimal key presses
  • Support for multiple language dictionaries
  • Classic mobile phone typing experience

History and Usage

In the late 1990s, T9 was used in phones by manufacturers such as Verizon, NEC, Nokia, Samsung, Siemens, Sony Mobile, Sanyo, and SAGEM, as well as PDAs like Avigo. Key competitors included Motorola's iTap, RIM's SureType, Eatoni's LetterWise and WordWise, and Intelab's Tauto.

Today, T9 is still available on niche devices like the Punkt mp-02, on some non-touchscreen phones, and as a custom keyboard for Android and iPhone devices (from iOS 18).

How T9 Works

  • Each numeric key (2-9) represents 3-4 letters
  • The system predicts words based on the sequence of key presses
  • It uses a dictionary to suggest the most likely words
  • Users can cycle through suggestions using the * or # keys
  • The algorithm uses a Trie data structure for efficient word lookup

Application Setup & Running

Requirements:

  • Python 3.x installed on your system
  • Basic understanding of command line/terminal

Clone the Repository:
git clone https://github.com/Maniekw12/T9-system.git

Navigate to Project Directory:
cd T9-system

Run the Simulator:
python t9_simulator.py

Technology Stack

  • Language: Python 3.x
  • Data Structure: Trie (Prefix Tree)
  • Algorithm: Predictive text matching
  • Dictionary: Custom word database

Use Cases

  • Educational tool for understanding predictive text algorithms
  • Nostalgia project for experiencing classic mobile phone typing
  • Data structures demonstration (Trie implementation)
  • Input method research and development

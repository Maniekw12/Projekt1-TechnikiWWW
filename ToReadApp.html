About ToReadApp

The application is created for people who want to track their progress in reading books. It allows you to create your 'To Read' list, where you can prioritize which book is the most important for you and manage which books are next in the queue to read. It also stores the books you have already read, so you can track how many books you have completed.

Build the Application:

Run the Application:

Stop the Application:

Once everything is set up and running, the application will be available at: http://localhost:8000
